Interactive-Editable-Gaussians: [IEG-TBD]

Junsuk (John) Kim1, Xin (Shane) Li1,
1Texas A&M University - College Station

[IEG-TBD] uses Google Earth Studio to create a 3D scene.

Abstract

We present a convenient and efficient method for creating free-viewpoint landscapes reconstruction

Our approach utilizes Google Earth Studio to collect data in aerial view. First, we draw a trajectory for the camera view and rotate around the desired object. After computing the distance and angle, we can create a trajectory for any object with given GPS coordinates.

We show that [IEG-TBD] can easily generate new 3D scenes with a few clicks. With rendering new view points with the computed camera trajectories, we can generate more image data to train the model for any object in the scene.
